React component for the /invite/:enterpriseCustomerInviteKey route.

Redirects user to enterprise logistration flow to create a new account or login to an existing account, then get redirected back to this route. At this point, we attempt to link the user to the EnterpriseCustomer associated with the enterpriseCustomerInviteKey.

If successful, the user is redirected to their enterprise customer's Learner Portal slug. If an error occurred, an appropriate error page is shown and an error is logged.

resetModalState()

We're passing a function to reset the MarkCompleteModal state to its initial state here. That way, the consumer of this component can call this function to reset the modal state. When the open prop on the Paragon Modal component changes to false, it doesn't trigger the onClose callback, another place where we reset to initial state. Because of this, passing the resetModalState function gives control to the consumer of this component to reset to initial state as needed.
